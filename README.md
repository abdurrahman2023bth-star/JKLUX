# JKLUX Ecommerce Website

This is a complete static ecommerce website for the JKLUX clothing brand.

## Pages

- `index.html` - homepage
- `products.html` - product collection
- `product.html` - product detail page
- `cart.html` - shopping cart
- `checkout.html` - checkout
- `about.html` - about us
- `contact.html` - contact page

## How to Open

Open `index.html` in a browser. No server or installation is required.

## Features

- Premium black-and-white responsive design
- Local product and lookbook images
- 12-product catalog across Oversized T-Shirts, Premium Shirts, Cargo Pants, and Jeans
- Product filtering
- INR pricing
- Product detail page with S, M, L, and XL size selection
- Product image placeholders
- Cart saved in browser storage
- Quantity updates and item removal
- Checkout and contact form confirmation states
